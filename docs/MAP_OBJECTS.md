## 1. BookObject — книга

Источник стихий. Игрок подходит, удерживает 1 секунду — рулетка из 2 вариантов.

**Параметры:**
- На арене всегда **3 книги** (см. `game_controller._spawn_books`).
- Модель `Spellbook.glb` парит + вращается (sin-функции в `_process`).
- `Area3D` mask=1 (игрок) — `body_entered` запускает удержание.

**Удержание:**
- При входе игрока: автоматически `start_hold()`. 
- Прогресс рисуется как **круговой шейдерный лоадер** под книгой.
- Если игрок выходит до 1с — `cancel_hold`.
- Через 1с — открывается `ElementPicker`.

**Респаун:**
- После использования книга пропадает.
- Через 2с спавнится в новой случайной точке через `NavigationServer3D.map_get_random_point()` (гарантирует достижимость).

## 2. ElementPicker — рулетка стихий

UI поверх игры. PROCESS_MODE_ALWAYS (работает на паузе).

**Логика выбора 2 вариантов:**
- Берутся **контр-стихии** живых врагов на арене (через `ElementTable.get_counter`).
- Дубли удаляются.
- Если контров < 2 — добавляются случайные стихии.
- Из получившегося пула выбираются 2.

**UX:**
- Клавиши: `A` (левая), `D` (правая), `Esc` (отказ).
- Кнопка отказа (×) — закрыть без выбора. Книга при этом всё равно тратится (по дизайну — за нерешительность).
- На выбранной стихии вызывается `player.pickup_element(element)`.

**Edge-кейс:** все враги одной стихии? Невозможно по правилу (см. `COMBAT_CORE.md` §1).

## 3. ZoneObject — зона на арене

`Area3D` + цилиндрический `CollisionShape3D` (радиус 0.8м).

**Параметры:**
- Лимит активных — 2 (FIFO через `ZoneLogic`, см. `COMBAT_CORE.md` §2).
- Не удаляются по таймеру — только при переполнении.
- `collision_mask = 2` (враги). `body_entered` → проверка эффекта через `ZoneLogic.check_effect`.

**Эффекты (в `body_entered`):**
- `COUNTER` (контр-зона) → `CombatLogic.try_apply_mark` → если новая метка, эмит сигнала.
- `MATCHING` (своя) → `try_apply_rage`.
- `NEUTRAL` → ничего.

**Earth-зона особый случай:**
- Дополнительно создаётся `NavigationObstacle3D` — враги обходят. Это превращает Earth-зону в физическое препятствие на навмеше.
- Удаляется вместе с зоной.

**Визуал:**
- Шейдер с цветом стихии, пульсация.
- `_lifetime_timer` поле существует, но **не используется** (зоны не имеют времени жизни) — TODO почистить.

## 4. Декор пола: Trapdoor

В `arena_view._populate_floor_decor` на свободных тайлах (0.1% шанс) ставится `Trapdoor.glb`.

- Без коллайдера, без бликов (`metallic=0, roughness=1`).
- `Y = 0` — часть пола, не закрывает индикаторы (которые на Y=0.25).
