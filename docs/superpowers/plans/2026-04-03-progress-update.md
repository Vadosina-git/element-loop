# Element Loop — Обновление прогресса (2026-04-03)

## Реализовано (Phase 1 — базовый геймплей)

### Игрок
- [x] CharacterBody3D: движение WASD, джойстик, tap-to-move
- [x] Рывок (Shift): дистанция 3.0, кулдаун 5 сек, частицы дыма
- [x] HP: 2, мигание красным при уроне
- [x] Постановка зон стихий

### Враги (x20)
- [x] 5 стихий, равномерный спавн по карте
- [x] AI FSM: Wander → Chase → Attack → Recover
- [x] Зона обнаружения с пунктирным кругом
- [x] Контакт-урон с кулдауном 1 сек
- [x] Значки стихий над головой (анимированные)
- [x] Обход препятствий при патрулировании

### Боевая логика
- [x] Таблица контров (5 стихий, циклическая)
- [x] Метки (контр-зона → 3 сек → 1 HP урона, замедление 25%)
- [x] Ярость (своя зона → +25% скорость, 4 сек)
- [x] Зоны: постановка, FIFO лимит, контр/своя/нейтральная

### Карта
- [x] Арена 36x54 с тайловым полом (KayKit, scale 0.25)
- [x] Стены по периметру (KayKit, scale 0.25)
- [x] Процедурный лабиринт из камней (100 гряд, длина 1-10)
- [x] Рандомная генерация при каждом запуске/рестарте

### Книги (x20, = количество врагов)
- [x] Случайный спавн, равномерно по карте
- [x] Автоисчезновение (8-16 сек) + мигание перед исчезновением
- [x] Респавн в новой точке
- [x] Удаление 1 книги при убийстве врага
- [x] Стрелки-индикаторы для книг за экраном

### HUD
- [x] HP (левый верх)
- [x] Слот стихии (центр верх) — эмодзи + название + цвет контейнера
- [x] Кнопка «Зона» + «Рывок» с заливкой кулдауна (правый низ)
- [x] Колесо стихий со стрелками контров (правый верх)
- [x] Пресеты камеры (центр низ)
- [x] Game Over экран + перезапуск (R)

### Камера
- [x] 4 пресета: Стандарт, Ближняя, Top-Down, Изометрия
- [x] Слежение за игроком (lerp)
- [x] Плавные переходы между пресетами
